
# אגם רולי – לומדים ומשחקים (PWA)
אתר אינטראקטיבי בעברית לילדים על Rolley Lake Provincial Park (Mission, BC), כולל משחקים, מפה (OpenStreetMap / Leaflet), ותמיכה בהתקנה כ־PWA.

## הפעלה
פתחו את `index.html` בדפדפן מודרני. כדי להתקין כ־PWA, פתחו את האתר דרך HTTPS (למשל דרך GitHub Pages), ולחצו "הוסף למסך הבית".

> הערה: אריחי המפה נטענים מהאינטרנט בזמן אמת. משחקים ותוכן עובדים גם ללא חיבור לאחר טעינה ראשונית.

## ספריות חיצוניות
- Leaflet 1.9.4 עבור המפה: CDN של unpkg

## קריינות
הכפתור "קריינות" משתמש ב־Web Speech API (speechSynthesis). התמיכה תלויה בדפדפן ובקולות המותקנים במערכת.

## מקורות ידע
- BC Parks – Rolley Lake Park
- City of Mission – Rolley Falls Trail
- AllTrails / VancouverTrails – מרחקים והסברים כלליים על מסלולים
